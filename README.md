# MainScene

## 캐릭터 선택

캐릭터 선택은 OnClick 이벤트로 진행되며, OnClick 이벤트를 호출하는 객체의 ScriptableObject를 GameManager에게 넘겨주게 되며, GameManager는 해당 ScriptableObject를 기반으로 Player를 초기화한다.
![캐릭터선택창](./.image/캐릭터선택창.gif)
